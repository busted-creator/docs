BustedSolver is a fast, reliable API for reCAPTCHA v3 solving and Square PCI tokenization. One API call per service. Simple auth, real-time usage tracking.

  <Card title="reCAPTCHA v3 Solver" icon="shield-check" href="/solve">

  </Card>
  <Card title="Square Tokenizer" icon="credit-card" href="/tokenize">
    Tokenize cards through Square PCI and get a card nonce with buyer verification in one call.
  </Card>
</CardGroup>

## Quick start

<CodeGroup>

```python reCAPTCHA v3
import httpx

r = httpx.post(
    "https://api.bustedcap.com/api/solve",
    json={
        "site_key": "6Le...",
        "page_url": "https://example.com/page",
        "action": "submit",
    },
    headers={"X-API-Key": "sk_live_your_key_here"},
    timeout=60,
)

data = r.json()
print(data["token"])  # reCAPTCHA v3 token
```

```python Square Tokenizer
import httpx

r = httpx.post(
    "https://api.bustedcap.com/api/tokenize",
    json={
        "cc": cc, "mm": mm, "yyyy": yyyy, "cvv": cvv,
        "app_id": "sq0idp-xxxxx",
        "loc_id": "LXXXXX",
        "base_url": "https://example.square.site",
    },
    headers={"X-API-Key": "sk_live_your_key_here"},
    timeout=60,
)

data = r.json()
print(data["card_nonce"])          # Square card nonce
print(data["verification_token"])  # Buyer verification token
```

</CodeGroup>

## Base URL

```
https://api.bustedcap.com
```

## Authentication

All requests require an API key passed in the `X-API-Key` header. See [Authentication](/authentication) for details.

```
X-API-Key: sk_live_your_key_here
```
