# BustedSolver API

BustedSolver is an API with two services: **reCAPTCHA v3 solving** and **Square PCI tokenization**. One API call per service. Fast, reliable, and simple.

## Why BustedSolver?

- **Fast** -- responses in seconds
- **Reliable** -- high-score reCAPTCHA tokens and full Square 3DS support
- **Simple** -- two endpoints, same auth, same API key
- **Usage tracking** -- monitor your remaining requests anytime

## Services

| Service | Endpoint | Description |
|---------|----------|-------------|
| [reCAPTCHA v3 Solver](/solve) | `POST /api/solve` | Solve reCAPTCHA v3 and get a valid token |
| [Square Tokenizer](/tokenize) | `POST /api/tokenize` | Tokenize a card via Square PCI + 3DS verification |

## Quick Start

### reCAPTCHA v3

```python
import httpx

r = httpx.post(
    "https://api.bustedcap.com/api/solve",
    json={
        "site_key": "6Le...",
        "page_url": "https://example.com/page",
        "action": "submit",
    },
    headers={"X-API-Key": "sk_live_your_key_here"},
    timeout=60,
)

data = r.json()
print(data["token"])  # reCAPTCHA v3 token
```

### Square Tokenizer

```python
import httpx

r = httpx.post(
    "https://api.bustedcap.com/api/tokenize",
    json={
        "cc": cc, "mm": mm, "yyyy": yyyy, "cvv": cvv,
        "app_id": "sq0idp-xxxxx",
        "loc_id": "LXXXXX",
        "base_url": "https://example.square.site",
    },
    headers={"X-API-Key": "sk_live_your_key_here"},
    timeout=60,
)

data = r.json()
print(data["card_nonce"])          # Square card nonce
print(data["verification_token"])  # Buyer verification token
```

All requests require an API key in the `X-API-Key` header.

```

```
