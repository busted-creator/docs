---
title: "Check Usage"
description: "Check your API key usage and remaining requests."
api: "GET /api/usage"
---

Check how many requests you have used and how many remain.

## Request

<ParamField header="X-API-Key" type="string" required>
  Your API key.
</ParamField>

## Example

<CodeGroup>

```python Python
import httpx

r = httpx.get(
    "https://api.bustedcap.com/api/usage",
    headers={"X-API-Key": "sk_live_your_key_here"},
    timeout=10,
)

print(r.json())
```

```bash cURL
curl https://api.bustedcap.com/api/usage \
  -H "X-API-Key: sk_live_your_key_here"
```

</CodeGroup>

## Response

<ResponseField name="owner" type="string">
  Key owner name.
</ResponseField>

<ResponseField name="requests_used" type="integer">
  Total requests made with this key.
</ResponseField>

<ResponseField name="requests_limit" type="integer or null">
  Maximum allowed requests. `null` if unlimited.
</ResponseField>

<ResponseField name="rate_per_minute" type="integer">
  Maximum requests allowed per minute.
</ResponseField>

<ResponseField name="expires_at" type="string or null">
  Key expiration date in ISO format. `null` if no expiry.
</ResponseField>

## Errors

<Accordion title="401 -- Invalid API key">

```json
{
  "detail": "Invalid API key"
}
```

Your API key is incorrect or does not exist. Double-check the key value.

</Accordion>
